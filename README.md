### Todolist Application
É uma lista de tarefas que você precisa concluir ou coisas que deseja fazer.

Baseado em: https://github.com/tastejs/todomvc

####  Funcionalidades:

- Criar todo ☐

- Listar todos ☐

- Completar todo ☐

- Excluir todo(s) ☐

- Excluir todo(s) completas ☐

- Contagem das todos ☐   

- Filtros
  - Todas ☐
  - Ativas ☐
  - Completas ☐
